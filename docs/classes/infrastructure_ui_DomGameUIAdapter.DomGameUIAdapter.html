    Class DomGameUIAdapter

    Adaptador de interfaz de usuario basado en el DOM.

    Implementa el puerto GameUIPort, proporcionando las operaciones necesarias para actualizar visualmente el tablero y el teclado del juego Wordle.

    Este adaptador es parte de la capa de infraestructura dentro de la arquitectura hexagonal, permitiendo que la lógica del juego interactúe con el entorno del navegador sin depender directamente del DOM.
